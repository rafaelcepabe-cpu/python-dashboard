## 🚀 Sobre o Projeto

Este projeto consiste em um **dashboard interativo** desenvolvido em Python utilizando Streamlit, com o objetivo de facilitar a análise de dados de forma visual e intuitiva.

A aplicação permite que o usuário explore os dados por meio de gráficos dinâmicos, filtros personalizados e métricas estratégicas, sem necessidade de conhecimento técnico.

---

## 🎯 Objetivo

Transformar dados brutos em **informações visuais e insights relevantes**, auxiliando na tomada de decisão.

---

## 🛠️ Tecnologias Utilizadas

* Python
* Streamlit
* Pandas
* Plotly
* Numpy

---

## ⚙️ Funcionalidades

✅ Upload de arquivos (CSV/Excel)
✅ Filtros interativos (data, categoria, etc.)
✅ Gráficos dinâmicos (linha, barra, pizza)
✅ Exibição de KPIs (métricas principais)
✅ Interface simples e intuitiva

## ▶️ Como Executar o Projeto

```bash
# Clone o repositório
git clone https://github.com/rafaelcepabe-cpu/python-dashboard.git

# Instale as dependências
pip install pandas, streamlit, plotly e numpy

# Execute o projeto
streamlit run dash.py
```
